# Cotizador de Seguros - Proyecto desde 0

Instrucciones abrir el proyecto final:

1. Abrí la terminal en la carpeta del proyecto.
2. Ejecutá `npm install` para instalar dependencias.
3. Ejecutá `npm run dev` para levantar el servidor de desarrollo.
4. Abrí en tu navegador la URL que te muestra Vite (ej. http://localhost:5173).
